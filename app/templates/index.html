What does the project do?

The expeditions dataset consists of information of the past expeditions in the himalayan region, such as the netionality of the expeditioners, the meximum elevevation reached, the seasion of expedition, etc... The goal of this project will be to train the model in a way that it can predict the requirement of O2 in a upcoming expedition based on the selected variables. Here the maximum elevation reached in any expedition is 8850 meters.

Machine Learning


Logistic Regression: Explanation

First the baseline logistic regression was applied. Further, since there is a scope of linear separation of data based on the max_elev_reached features as evident from 'O2 used VS Max elevation reached' graph in the EDA section, this might as well be the final model after comparison with other models.

Logistic Regression: Results

Below is the confusion matrix from the logistic regression model.

From the above confusion matrix it can be observed that the false negative value using the logistic regression is 111, which in turn converts to a recall of 81.46%. Which is decent considering the fact that for a prediction of O2 for an expedition we need the recall to be as high as possible to avoid any deaths because of incorrectly judged requirement of O2.

Random Forest: Explanation

An ensemble technique to ensure the best results. Though, with this method, it is necessary to keep a check on overfitting which we will consider, looking at the train and test accuracies of the model.

Random Forest: Results

Below is the confusion matrix from the random forest models.

From the above confusion matrix it can be observed that the false negative value using the random forest is 149, which implies thatthe recall is 75%. This recall value is comparatively low when compared to that of the logistic regression and hence use of this model as the model of this prediction is not recommended. Further, as the comparison table shows, random forest is not generalizable given the vast different between the test and train accuracy, indicating overfitting.

SVM: Explanation

Support Vector Machine (SVM) is another linear technique applied to this dataset. SVM is known to be quick learner with less data to train on.

SVM: Results

Below is the confusion matrix from the support vector maching model.

Unfortunately, the SVM model follows the similar recall rate as that of the random forest above i.e., 75%. This makes is a weak candidate for this project as the recall rate is relatively low when compared to logistic regression.

Neural Network: Explanation

Neural networks are one of the most powerful machine learning approaches for data with complicated patterns, but are more prone to overfitting to simple data.

Neural Network: Results

Below is the confusion matrix from the Neural Network model.

The Neural Network model is the second candidate that can be used for this project, given the fact that it has the highest recall rate of 84.3% with the false negative value as 94 (lowest in the 4 models). Though this is a strongest candidate in the 4 models, the accuracy values and F1 score is the lowest in the 4.

Conclusion


In conclusion of the project findings, we can clearly see based on the recall and false-negative scores that Neural Network and Logistic Regression lead the model generalizability along with good accuracies of 87.8% and 90.2% respectively. Further, it is known that Neural Networks, are strong predictors of models where the training data is large (in this case 8269). However, models based on neural networks have relatively low interpretability in comparison to the simpler linear models (such as Logistic Regression and SVM). In this case, considering the recall score, the neural network model stands at the top. It is very important in cases of high altitude mountaineering to have O2 stocked up in case it is required 7K m plus above sea level and hence despite a slightly lower accuracy, when compared to other models, for this task we will move forward with the neural network as our model of choice. As a plausible extension to this project, it would be a plus point for the users using this product to get information about 1. dynamic data addition using the lastest predictions to help the model learn as it predicts 2. tune the hyperparameters of the Neural Network model to achieve better results

So what?

So what is the significance of our project? Since we are aware of the best features needed to predict the target "is_o2_used" based on domain knowleged and Neural Network as the model of choice, we can efficiently predict the requirement of O2 in a future expedition.
